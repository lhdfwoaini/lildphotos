## 更新原则

1. 只将适合公开的优化副本放入 `photos/` 对应栏目目录。
2. 在 `index.html` 的 `seriesList` 中维护栏目与作品数据。
3. 所有图片使用 `photos/...` 相对路径。
4. 不要删除 `.nojekyll`；配置自定义域名后不要删除 `CNAME`。
5. 不上传原始图库、隐私信息、密钥或本地开发缓存。
